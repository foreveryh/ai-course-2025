# W2 作业模板 · Prompt 模板（RACER）

使用说明：复制下方模板，按任务实际情况补充，每一项尽量具体与可验收。

---

## 1) Role · 角色
- 你希望模型扮演的专业身份：
- 能力边界（可做/不可做）：
- 目标受众（面向谁）：

## 2) Aim · 目标
- 产出目标（做什么）：
- 成功标准（可度量/可检验）：
- 约束（长度/语气/风格/格式/禁忌）：

## 3) Context · 上下文
- 背景材料（原文/摘要/要点）：
- 关键实体与指标（避免丢失）：
- 相关链接或附件：

## 4) Examples · 示例
- 正例（你想要的风格/结构）：
- 反例（你不想要的）：
- 对比说明（为什么好/不好）：

## 5) Rules · 规则
- 输出结构（JSON/表格/要点）：
- 步骤（先…再…最后…）：
- 容错（允许“未知/不可得”字段为空，不得编造）：

---

## 自检（Self-Refine）提示片段
- 列出本次输出可能的3个风险点，并逐条修正后给出修订版。
- 对照验收清单，标注未满足项并重写。
