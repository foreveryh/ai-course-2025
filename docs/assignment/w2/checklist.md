# W2 作业模板 · 验收清单（Checklist）

使用说明：按内容/形式/过程三大维度检查，标注“必改项”。

## A. 内容（Content）
- [ ] 技术/事实准确（必改）
- [ ] 目标达成（与Aim一致）（必改）
- [ ] 覆盖完整，无关键遗漏（必改）
- [ ] 逻辑自洽，有依据可追溯

## B. 形式（Form）
- [ ] 长度/语气/风格符合约束
- [ ] 输出结构符合模板（字段齐全、类型正确）（必改）
- [ ] 禁忌遵守（不得编造/夸大）（必改）
- [ ] 可读性良好（标题/要点/层级清晰）

## C. 过程（Process）
- [ ] 按步骤执行（描述→分解→产出→自检）
- [ ] 提供中间结果与思路（必要时）
- [ ] 进行了自检（Self-Refine）并说明修订点
- [ ] 可复现（上下文与规则可被他人复用）

---

评分建议：
- 采用 1-5 分 + 必改项列表。
- 必改项未达标 → 退回重做。
